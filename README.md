Make links!
-----------

Make links in ~/ for the following files/dirs:

* .gitconfig
* .vimrc
* .fonts/
* .vim/
* and the **dotfiles/* directory!


oh-my-git
---------

Don't forget to:

1. install fonts: `# fc-cache -fv ~/.fonts`
2. add the following line to the `.bashrc` file: `source $HOME/dotfiles/oh-my-git/prompt.sh`
